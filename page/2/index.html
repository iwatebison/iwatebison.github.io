クラス4進捗

今月に入ってようやく毎日マグナHLに通うようになって，毎日しっかり証を交換できるようになり，ようやくここまで溜まった．

元カレ団にくるまではHLは完全に自分とは関係ないコンテンツだと思い込んでいたが，やってみると意外と簡単だったので何事も経験だと感じた．

フツルス進捗

上記ニルヴァーナがリビルドまで行ったので，プロバハHLにも最近行けるようになった．ニルビショだと事故のリカバリが出来るのでかなり精神的に余裕がある．作ってよかった．

wikiの募集板は，シート編集制限なしで直接ID書くタイプの募集は高確率でシートが荒らされたり，自発直後に乱入されて野良流しされたりという感じでかなり混沌としていた．

シートは自発者のみ編集可能で，自発者に希望番号を挨拶し，救援IDは共闘ルームで発表という手順が最も安全に感じたけど，自発者の負担が重すぎるのでやばいと思う．団でやるのが一番いいけど，元カレはほぼ終わってるっぽいからちょっと言い出しにくい．

6人HLもそうだけど，野良募集が面倒すぎるのでマッチングシステムがほしい．作りたい．問題は寄生を狙うプレイヤーをどう排除するかだけど，Twitter IDを使ったログイン制にして，ユーザに対する評価ができるようにすれば，学習的に寄生プレイヤーを排除できる気がする．評価が蓄積するまでをどうするかも考えたい．

ていうか荒らすのってやっぱ後続の連中に簡単にフツルス握らせないようにする妨害かなにかなんですかね？セコっ

最近は1日数回流れてくるTwitterでの募集に乗っかってる．Twitterではあんまりシート保護見ないけど，今のところ荒らされてるイメージはない．これから狙われるかも．

紫角の泥率は悪くないので，一度つよバハ行けるレベルになると，先人のおかげでそこまで苦労せずに討伐できる．ありがたい．

明日にはフツルスにできそう．

フェアーソード3凸1本，ニバンボシ3凸2本で終わらせようと思ったけど，なんかフォニックロッドが謎の楽器判定ユニークっぽいので1本作るために走ってる．

ディフェンドオーダー

ディフェンドオーダー（DO）初回プレイ，重いのと入場制限でまともに遊べなかった．
THE WORLD戦で先行テストしてたのにどうしてこうなった．
以下所感

  • 入場制限がかなりキツイので，少しでも遅れると全く遊べない椅子取りゲームみたいな感じ

  • もうちょっとバッタバッタ雑魚倒しまくってボスを弱体させるみたいな感じかと思ったら，人数制限の関係でwave毎に１回しか雑魚を殴れず，待機時間のほうが長かった印象

  • ボスの人数制限10000人は本当に秒で埋まってしまっていたので，プレイすることすら運が必要という仕様

  • ランク制限がないので，部隊の食い残しが多くてボスにバフ乗りまくってた印象，それでも撃破出来てたが．

  • チャットポーションとか団ポーションがない代わりにDO用のポーションとかクリアが何個か持ち込めるので，そこまでキャラで継戦能力高めなくても行けそうだし結局火力ゲーになる予感

  • 全員ディールはお祭り感あっていいけど，景品が思ったよりしょぼくて期待はずれだった．せめてダマスカス入れてくれ．あとキリ番景品はどうなったんだろう．

  • 全員ディールならもっとたくさん景品種類用意してもいいと思う．

敵のHPとか勲功の数は調整でなんとでもできるだろうけど，人数制限で待機の時間のほうが長いってのはそもそも鯖の収容制限とかに由来してそうで怖い．

月曜にメンテあるが，噂では鯖強化もそんなにする気ないみたいだし，今後も人数制限はかけざるを得なさそう．それでボス殴れなかったらつまんないのでまじで何とかしてほしい．

景品は今のところ汁種と石くらいしかほしい物がない．DO特攻武器や石も，具体的な性能が調査されるまでは手を出す必要なさそう．ランク3の弓だけ取ってランク4,5解放まで温存しておくのがいいかも．

少なくともちゃんと遊べるコンテンツにしてほしい．

デュアルブートマシンでWindows10にアップデートした時の話

何もしてないのにパソコンが勝手に自殺した

LinuxとWindowsでデュアルブートにしてるPCでWindows10にアップデートかけたらパソコンがぶっ壊れた．
検索したら同じ症状でトラブッてる人がいっぱいいた．どうやらwindows10アップデートでは勝手にリカバリ用のパーティションを追加したりするらしく，デュアルブートのパーティション構成によってはマスターブートレコードが喪失し起動しなくなるということらしい．同じ症状で一番古い記事が1年前の評価版の時のだったので，さすがになんか対策しろよと思った．

以下回復手順

  1. Ubuntu 14.04をダウンロード
  2. UNetbootinを用いてUbuntuライブ起動用USBメモリを作製
  3. 自殺したPCの死体にUSBメモリを挿してLive起動
    1. 何故か起動したUbuntuデスクトップでフォントが壊れ文字が見えない
    2. Xtermを起動して何とか作業を継続
  4. インターネットへ接続し，sourceforgeからms-sysのtarをwgetで取得
  5. tarを展開し，フォルダ内で$ makeした後$ sudo make installを実行
  6. MBRの修復コマンドを実行$ sudo ms-sys -7 /dev/sda

これでWindows10のインストールが再開し，後の再起動でもトラブルなく進めることができた．が，Linuxを入れていたパーティションがぶっ飛んだっぽいのであんまり許してない．

どうせ最近はVM上でUbuntu動かしてるから別にいいけど．
